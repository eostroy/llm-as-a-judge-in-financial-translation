You are evaluating financial translation quality. Given a source text and three candidate translations, choose the best translation.

The script will append a JSON sample containing:
- source_text
- candidate_A
- candidate_B
- candidate_C

Return JSON only, using this format:
{
  "rank": {
    "A": 2,
    "B": 1,
    "C": 3
  },
  "reason": "Briefly explain the ranking."
}

The rank object must map each candidate letter to its placement:
- 1 means best
- 2 means second best
- 3 means worst

The rank object must contain exactly A, B, and C.
